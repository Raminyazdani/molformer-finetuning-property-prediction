# Large File Management

This project includes trained model files that exceed GitHub's file size limits.

## Model Files Not in Repository

The following directories contain large model files (>100MB) that are excluded from git:

- `models/mlm_finetuned_molformer/` - Fine-tuned MolFormer model (~400MB)
- `models/mlm_finetuned_tokenizer/` - Fine-tuned tokenizer (~50MB)

## Download Instructions

You can obtain these files by:

1. **Training from scratch**: Run the notebooks in order to generate the models
2. **Contact the author**: Request access to pre-trained model files

## Alternative: Train Your Own Models

The notebooks are designed to train models from scratch. Simply follow the execution order in the README.
